При запуске ```python folder/subfolder/file.py``` 
путь для ```import``` интерпретируются относительно главного запускного файла (```__main__```).
Не важно откуда запускается, важно где лежит файл.

При запуске ```python -m folder.subfolder.file``` 
путь для ```import``` интерпретируются относительно папки откуда запускается команда.
